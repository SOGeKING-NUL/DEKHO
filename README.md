# DEKHO
This project builds an AI-powered system to track and geolocate individuals across multiple CCTV cameras in real time. Using YOLOv8 for detection, DeepSORT for tracking, and OSNet for Re-ID, it identifies suspects and maps their movements on a live dashboard (React + Leaflet.js).
